DESCRIPTION:
-------------------------------------------------------------------------------

Enabling the ability for multiple products to be added to the wishlist at the 
same time.


-------------------------------------------------------------------------------
HOW TO USE:
-------------------------------------------------------------------------------

Supposedly, this is already used in external systems and code.


-------------------------------------------------------------------------------
EXTENSION FILES:
-------------------------------------------------------------------------------

  - app/code/local/CSH/Wishlist/*
  - app/etc/modules/CSH_Wishlist.xml
  

-------------------------------------------------------------------------------
COMPATIBILITY:
-------------------------------------------------------------------------------

  - Tested with Magento EE 1.12.0.0
